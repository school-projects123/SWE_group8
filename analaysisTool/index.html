Explanatory Text

As per this version of the special branch the data from the xls(x) files can be read in by the javascript, which creates a table and a chart from the data (shown in this web-page), where the scores are shown with bars, the highest score in green, lowest in red, and the rest in yellow.

In order to reference the old app file and my old sample data, you can check the git history (as of this commit specifically).

At present this is entirely a proof of concept and will not work with any differently formatted data, different column names, or pretty much anything else. However if another file shares column names with the sample data any matching columns will be used.

Note: if you typed something into the prompt and it showed nothing, press enter in the text box (so it uses the default) or copy and paste '21755561_Pretend_Essay_Assignment.xls' into the prompt. You can also try 'gc_PS5906_25_SANDPIT_fullgc_2025-10-07-13-12-38.xls' to compare.
